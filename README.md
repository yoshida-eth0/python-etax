## 所得税と住民税の計算をするプログラム。

### 生成できるもの
- 所得税青色申告決算書 損益計算書
- 所得税及び復興特別所得税の申告内容確認表 第一表
- 特別区民税・都民税・森林環境税 課税証明書
- ふるさと納税の控除上限額

### 実装状況
##### 所得税 -> 総合課税
営業等所得のみ実装。

##### 所得税 -> 総合課税 -> 所得控除
社会保険料控除、小規模企業共済等掛金控除、基礎控除、寄附金控除(ふるさと納税)のみ実装。

##### 所得税 -> 総合課税 -> 税額控除
未実装。

##### 所得税 -> 分離課税
未実装。

##### 住民税 -> 所得控除
社会保険料控除、小規模企業共済等掛金控除、基礎控除のみ実装。

##### 住民税 -> 税額控除
調整控除、寄附金控除(ふるさと納税)、定額減税のみ実装。
